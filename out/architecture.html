システム概要

このシステムは以下のコンポーネントで構成されています。

コンポーネント構成

フロントエンド

  • React.js ベースのユーザーインターフェース
  • TypeScript による型安全性

バックエンド

  • Node.js + Express サーバー
  • REST API エンドポイント

詳細なAPI仕様は API仕様 を参照してください。

データベース

  • PostgreSQL によるデータ永続化
  • Redis によるキャッシュ機能

セキュリティ

  • JWT トークンによる認証
  • HTTPS 通信の強制
  • CORS ポリシーの適用
