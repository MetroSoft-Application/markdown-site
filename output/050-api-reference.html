050-api-reference

🔧 API仕様

このドキュメントでは、REST API の詳細仕様を説明します。

ベースURL

https://api.example.com/v1

認証

すべてのAPIエンドポイントはJWTトークンによる認証が必要です。

Authorization: Bearer {your-jwt-token}

エンドポイント一覧

ユーザー管理

GET /users

ユーザー一覧を取得します。

GET /users

レスポンス例:

{
  "users": [
    {
      "id": 1,
      "name": "太郎",
      "email": "taro@example.com"
    }
  ]
}

POST /users

新しいユーザーを作成します。

POST /users
Content-Type: application/json

{
  "name": "花子",
  "email": "hanako@example.com"
}
